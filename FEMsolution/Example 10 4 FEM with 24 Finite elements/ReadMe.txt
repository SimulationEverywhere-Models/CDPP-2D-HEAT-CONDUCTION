Example 10.4 in 
-Tirupathi R. Chandrupatla, Ashok D. Belegundu, “Introduction To FINITE ELEMENTS in ENGINEERING”, Prentece Hall, 1997.

Program used form diskette with the previous reference.

Program name: HEAT2D2.BAS
Input file:   HEAT2D2N.INP
Output file:  OUTPUT.TXT


How to repeat the simulation:

1- in DOS screen from Accessories, change to the directory of the example.
2- type the command "QBASIC", then press "ESC".
3- File--> Open, then select the program HEAT2D2.BAS.
3- Run --> Start, to execute the program.
4- Enter names of Input File, Output File, other info is not important for this example.
